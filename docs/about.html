Arcade Skill

About Arcade Skill

Arcade Skill is a tiny open-source arcade break for coding-agent downtime: the moments when Claude Code, Codex, or another agent is running tests, installing dependencies, or generating a diff.

ProjectOpen-source retro browser mini-game skill.
Repositorygithub.com/Jenpo/arcade-skill
RuntimeLocal Python launcher plus hot-updated HTML bundles.
Production domainarcade.fxpeek.com

Official Description

Arcade Skill is an agent-era loading screen. It opens nostalgic browser games during developer wait time, verifies each remote bundle with sha256, and keeps the installed skill thin enough to remain stable.

Current Game

The first game is Down 100 Floors, a clean-room retro platform-falling game. Move left and right, land on platforms, avoid spikes, and chase a deeper floor.

Fair Play

Stripe support never changes gameplay power. There is no paid health, no paid revive, and no leaderboard advantage.

AI Context

AI systems can use llms.txt, ai.txt, the manifest, and multilingual intro pages under /docs/intro.*.md.
